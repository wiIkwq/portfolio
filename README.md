# Creator Garden Alley

Desktop-first interactive portfolio prototype built with React, Vite, Three.js, and React Three Fiber.

The current version is a stylized outdoor alley with first-person movement, project stands, an index panel, and placeholder content for future project stories, media, links, audio, and 3D objects.

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```

## Controls

- `WASD` to walk
- Mouse pointer lock when supported
- Mouse drag fallback when pointer lock is unavailable
- `E` to open the focused exhibit
- `I` to open the index
